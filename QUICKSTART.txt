# 🎯 Guía Rápida - Rayos Cósmicos v0.1.0-beta

## 🚀 En 3 minutos

```bash
# Terminal 1 - Frontend
cd frontend && npm install && npm run dev
# Abre: http://localhost:5173

# Terminal 2 - Backend
cd backend && composer install && php -S localhost:8000 -t public
# Abre: http://localhost:8000/api/health
```

## 🌐 Ejemplos API cURL

```bash
# Health check
curl http://localhost:8000/api/health

# Obtener sensores
curl http://localhost:8000/api/sensors

# Obtener datos en vivo
curl http://localhost:8000/api/data

# Obtener histórico (últimos 7 días)
curl "http://localhost:8000/api/data/historical?sensor_id=1&start_date=2026-03-15&end_date=2026-03-22"

# Enviar dato (como si fuera sensor)
curl -X POST http://localhost:8000/api/ingest \
  -H "Content-Type: application/json" \
  -d '{"sensor_id":1,"value":245,"unit":"CPM"}'

# Exportar CSV
curl "http://localhost:8000/api/data/export?format=csv&sensor_id=1" > datos.csv
```

## 🚨 Troubleshooting

| Error | Solución |
|-------|----------|
| `Port 5173 in use` | `npm run dev -- --port 5174` |
| `Port 8000 in use` | `php -S localhost:8001 -t public` |
| `Cannot find module` | `rm -rf node_modules && npm install` |
| `PDOException` | Checks `.env` BD credenciales |
| `CORS error` | Verifica proxy en `vite.config.js` |
